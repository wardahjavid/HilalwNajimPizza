## **Project Description**
HilalwNajimPizza is a fully-functional Java console-based Point-of-Sale (POS) system designed for a custom pizza shop.
This application allows employees to:
- Create and manage customer orders
- Build custom pizzas with toppings, crusts, sauces, and stuffed crust
- Select signature pizzas
- Add drinks and garlic knots
- View/filter/sort current orders
- Generate detailed receipts
- Log sales to daily reports

## **User Stories**
***Core User Stories***
- As a cashier, I want to create new customer orders so that I can check out customers efficiently.
- As a cashier, I want to customize pizzas so that I can fulfill different customer requests.
- As a cashier, I want to add drinks and garlic knots so I can complete full customer orders.
- As a cashier, I want to review my current order so that I can check accuracy before checkout.
- As a manager, I want a daily sales summary so that I can track total sales and number of orders.
- As a cashier, I want immediate feedback when I enter invalid inputs so that I can correct them quickly.
- As a user, I want clear menus and navigation so that I always know what to do next.

## **Stretch / Bonus Stories**
- As a cashier, I want to filter and sort the current order items so that I can find items quickly.
- As a manager, I want receipts automatically saved so that records are preserved digitally.
- As a manager, I want logs updated automatically so that I can audit sales history.

## **Running the Application in IntelliJ**
Follow these steps to get your application running within IntelliJ IDEA:
- Open IntelliJ IDEA.
- Select "Open" and navigate to the directory where you cloned or downloaded the project.
- After the project opens, wait for IntelliJ to index the files and set up the project.
- Find the main class with the public static void main(String[] args) method.
- Right-click on the file and select 'Run 'YourMainClassName.main()'' to start the application.

## **Technologies Used**
Technology	Purpose
Java 17: Core language for all logic
OOP / Clean Architecture: Layer separation (UI, Domain, I/O, Services)
File I/O (NIO / Writers):	Receipt + daily logs
Java Streams:	Filters & sorting
ANSI Escape Colors: Styled console output
JUnit 5:Unit testing
